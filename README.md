# 概要
系列文章整理于侯捷老师视频课程《C++新标准C++11/14》，在此感谢老师的辛勤付出。
视频中不全面但较为深刻的整理了 C++11/14 的部分重要新特性，包括语言和标准库两部分，尤其是对新特性背后的原理进行分析。
